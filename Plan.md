**1. Цели тестирования**

-  Написание чек-листа
-  Написание тест-кейсов
-  Тестирование функционала приложения
-  Покрытие приложения автотестами


**2. Типы тестирования**

-  Исследовательское тестирование
-  Функциональное тестирование
-  UI-тесты

**3. Инструменты для тестирования**

-  Android Studio - среда для написания тестов
-  Espresso — фреймворк для написания UI-тестов
-  Allure — фреймворк для составления отчетности о тестировании

**4. Критерии успешного завершения тестирования**

-  Приложение полностью покрыто автотестами
-  Все баги исправлены
